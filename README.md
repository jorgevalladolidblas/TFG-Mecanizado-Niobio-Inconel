# Anexos: MECANIZADO DE MATERIALES APORTADOS MEDIANTE LMD Y SU APLICACIÓN EN LA AERONÁUTICA

**Trabajo de Fin de Grado en Ingeniería**
**Autor:** Jorge Valladolid Blas

Este repositorio actúa como anexo digital complementario a la memoria del Trabajo de Fin de Grado. Contiene todo el material gráfico, archivos de código e informes metrológicos generados durante el proyecto.

## Índice del repositorio

* **📁 01_Codigo_Python:** Scripts desarrollados para la extracción y tratamiento automatizado de los datos de rugosidad.
* **📁 02_Planos:** Planimetría de las probetas de Niobio C103 e Inconel 718.
* **📁 03_Códigos_CAM:** Códigos del fresado multieje empleados en el centro de mecanizado.
* **📁 04_Metrologia_Alicona:** Informes originales de desgaste de herramientas y topografía tridimensional (valles, picos, curtosis) medidos con el sistema óptico Alicona.
* **📁 05_Catálogos:** Catálogos de las herramientas de corte
* **📁 06_Normas_metrología:** Normas empleadas a la hora de realizar las mediciones
